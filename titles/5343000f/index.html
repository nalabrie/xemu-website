Performance / Gameplay:
-----------------------
This game runs with slowdown running at 8fps(30fps game) for, most of the time I played. Should run at full speed with better hardware.
https://www.youtube.com/watch?v=rg-0q3h2xwA

Graphics:
-----------------------
The game is missing dynamic shadows for the character models, as well as some texture glitches that make some objects too bright or dark in a given scene. Also, the Blur filter is too strong in hell and causes issues when summoning to go to hell, such as artefacts on the top of the screen margin.

Overall most likely playable, but for anyone who completes the game, please update this report.
