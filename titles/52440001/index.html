compat_comments: Intro's / Cutscenes:
-----------------------
The FMV Intro is broken on my machine, flashing from the Xbox boot logo to the FVM repeatedly near the end of the FMV. This also occurs when using render scaling on the FMV.

Sound:
-----------------------
The sound seems to have a mixing issue with the announcer's voice sometimes cutting out in the menus but is very rare.

Performance / Gameplay:
-----------------------
This game runs at full speed.

Graphics:
-----------------------
The game suffers from no graphical glitches in the menus. In-game, the training instructor shadow is rendered incorrectly, creating artefacts on the floor mat with the shadow not being rendered, which affect all levels. On the dojo level, this also creates a wrongly placed shadow in the background, which moves with the character, but these are minor issues. 
Links to pictures of the bugs https://imgur.com/a/ZnLGTdM

I didn't Island Paradise, Desert Spring Resort and Alpine Retreat maps but this game is in a playable state.
